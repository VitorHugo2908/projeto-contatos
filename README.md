# projeto-contatos



Objetivo do projeto:

&nbsp;	Ser uma aplicação que liste os contatos de uma pessoa (nome e email) para facilitar a comunicação e listar todos os participantes de um determinado grupo/empresa. A aplicação além de listar os contatos oferece a possibilidade de editar, excluir e adicionar novos contatos.



Ferramentas utilizadas:

&nbsp;	As ferramentas utilizadas para essa aplicação foram Ionic, Firebase, API, Typescript.
